# 🔔 Jubilant-Invention: Ntfy Git Push Automator

A profile-aware notification system for Termux and Linux. Get rich, actionable mobile notifications every time you push code.

## ✨ Features
* **📍 Profile-Based:** Automatically detects project folder names.
* **🔗 Auto-Mapping:** Generates direct links to GitHub commit diffs.
* **📄 File Inventory:** Shows which files changed in the notification.
* **✨ Smart Updates:** Built-in version checking and changelogs.

## 📦 Installation
Copy and paste this into Termux:
\`\`\`bash
curl -sL https://raw.githubusercontent.com/jvhn0gl3/jubilant-invention/main/setup.sh | bash
\`\`\`

## 🚀 Usage
Inside any Git repo, just run:
\`gpush\`
